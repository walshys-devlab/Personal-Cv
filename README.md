<h1 style=" text-align: center;">📄 Personal CV Website</h1>

A responsive personal CV website built using HTML and CSS, designed to showcase my skills, experience, and projects as an aspiring Frontend Developer.

## 🚀 Overview

This project is a structured, semantic CV webpage that highlights:

Personal information
Technical skills
Work experience
Education
Projects

The layout uses CSS Grid for structure and focuses on clean UI, readability, and accessibility.

## 🛠️ Tech Stack

Technology	Purpose
HTML5	Semantic structure
CSS3	Styling and layout
CSS Grid	Page layout system
Flexbox	Component alignment
